📱 Friendship Tracking
Friendship Tracking is a personal relationship management platform designed to help users maintain meaningful connections with their friends and loved ones. By tracking the time since the last contact, it ensures you never lose touch with the people who matter most.

🔗 Live Link: https://friendship-tracking.netlify.app/

📝 Project Description
In a busy world, it is easy to forget the last time we reached out to a friend. This project solves this by providing a centralized dashboard to log interactions, set connection goals, and visualize relationship health through data analytics.

🚀 Technologies Used
Frontend: React.js (Vite)

Styling: Tailwind CSS & DaisyUI

Data Visualization: Recharts

Routing: React Router DOM

Notifications: React Hot Toast

Storage: LocalStorage for data persistence

🌟 Key Features
📊 Friendship Analytics: Features a dynamic Pie Chart (Donut style) that visualizes the distribution of your interactions—Calls, Texts, and Videos.

🔍 Advanced Timeline Filtering: A dedicated timeline page that logs every interaction, allowing you to filter by "Call", "Text", or "Video".

⏰ Interaction Tracking & Status: Automatically calculates "Days Since Contact" to show if your relationship is "On-Track" or "Overdue".
